Инструкция по локализации JoyShockMapper с помощью скриптов
Подготовка
    Убедитесь, что у вас установлен Python (у меня v3.13)

Шаг 1: Извлечение строк для перевода
    Запустите скрипт 1.extract-lines.py
		python 1.extract-lines.py

    Скрипт создаст два файла:
        extracted-lines.txt - содержит все найденные строки, требующие перевода
        extracted-lines_log.txt - содержит информацию о происхождении каждой строки (файл и номер строки)

Шаг 2: Подготовка файла перевода
    Откройте extracted-lines.txt в текстовом редакторе

    Внимательно проверьте все строки и удалите:
        Ошибочно добавленные строки
        Захардкоженные значения, которые не требуют перевода

    Сохраните отредактированный файл как translation-en.txt

Шаг 3: Перевод строк
    Создайте копию файла translation-en.txt с именем translation-xx.txt, где xx - код языка (например, ru для русского)

    Переведите все строки в файле translation-xx.txt, соблюдая:
        Исходный порядок строк
        Не изменяя форматирование
		Количество строк в обоих файлах должно быть одинаковым

Шаг 4: Синхронизация пробелов
    Откройте скрипт 2.sync-spaces-in-lines.py в текстовом редакторе

    Измените строку вызова функции, указав ваши файлы:
		sync_spaces('translation-en.txt', 'translation-xx.txt')

	Запустите скрипт:
		python 2.sync-spaces-in-lines.py

    Скрипт автоматически добавит правильные пробелы в начале и конце строк перевода, чтобы они соответствовали оригиналу

Шаг 5: Применение перевода
    Откройте скрипт 3.translation.py в текстовом редакторе

    Измените переменную с именем файла перевода:
		translated_file = os.path.join(script_dir, 'translation-xx.txt')

	Запустите скрипт:
		python 3.translation.py

    Скрипт:
        Создаст резервные копии исходных файлов
        Применит перевод ко всем соответствующим файлам
        Сохранит переведенные файлы в папке translated_files
		Создаст translation_log.txt, в котором указаны все замены и неиспользованные строки

Завершение
    Вручную переместите все переведенные файлы из папки translated_files в папку исходного кода JoyShockMapper, заменив оригинальные файлы

    Соберите проект обычным способом
